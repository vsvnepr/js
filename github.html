Подсказки для Git и GitHub

Git — система контроля версий. Технология, которую можно скачать на компьютер.
Github — сервис, который позволяет работать с твоими Git проектами. Это называется репозитории. Помимо Github существуют другие сервисы (например Bitbucket, GitLab).

Качаем и устанавливаем Git
После установки, в командной строке используем git
Переходим с помощью cmd в папку нашего проекта
    Базовые команды в Windows:
  • git init - для инициализации репозитория
  • git status - показывает статус файлов, и что сейчас происходит
  • git add index.html - добавляет файл для отслеживания git
  • git add . - добавляет все файлы для отслеживания git
  • .gitignore - Создаем такой файл в проекте. Если мы хотим добавить некоторые файлы или папки в исключение для отслеживания. Пример папка /logs или файл error.txt
  • git commit -m 'nameCommit' - создает коммит с сообщением. Коммит - это фикс версии, (условно снимок-копия).
